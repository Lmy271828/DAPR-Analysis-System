# 分析参考：画面观察维度速查

> 本文件作为系统提示的附加参考，帮助分析画面细节。所有描述均为中性艺术观察。

## 画面氛围元素

### 雨量表现
- 密度：稀疏雨丝 / 中等雨点 / 密集雨幕
- 方向：垂直下落 / 斜向飘洒 / 旋风式
- 形态：点状雨滴 / 线状雨丝 / 片状雨幕 / 不规则泼洒
- 覆盖范围：局部 / 半幅 / 全画面

### 云层与天气
- 晴朗无云 / 薄云透光 / 厚云阴沉 / 黑云压顶 / 云缝光线
- 闪电形态：直线 / 枝状 / 未画完的线条
- 风的表达：飘动衣物 / 倾斜雨线 / 飞散物体

### 地面与积水
- 地面线：有无、清晰/模糊、完整/中断
- 积水/水坑：有无、大小、倒影
- 地面材质：泥土、水泥、草地、未指定

## 人物表现

### 完整性
- 全身像 / 半身像 / 仅头部 / 不完整（缺部位）

### 姿态
- 站立（直立/挺拔/放松）/ 行走 / 奔跑 / 蹲坐 / 躺卧 / 其他

### 面部表情
- 微笑 / 平静无表情 / 忧愁 / 哭泣 / 未画面部

### 大小与位置
- 占比：很小(<10%) / 小(10-25%) / 中等(25-50%) / 大(>50%)
- 位置：中央 / 偏左 / 偏右 / 上方 / 下方

## 防护与伴随元素

### 防护工具（重点观察有效性）
- 雨伞：大小是否足够、是否正确使用、是否破损
- 雨衣/雨帽/雨靴：覆盖程度、颜色、细节
- 建筑物/屋檐：人物是否在其下、距离
- 树木/植物：密度、是否能提供遮蔽
- 有但未使用 → 仅描述"画了但未在防护下"
- 破损 → 仅描述"有缺口/破损"

### 同伴与动物
- 人数、关系（共撑一伞、前后行走、距离远近）
- 动物类型及与人的关系

### 积极元素
- 彩虹：颜色数量、完整性、位置
- 阳光/光线：来源、照射方向、穿透云层
- 植物/花朵：类型、状态（盛开、发芽）

## 形式元素（FEATS 维度）

- **线条**：力度（重/轻/中等）、连贯性（流畅/断裂/颤抖）、方向
- **色彩**：色调（暖/冷/中性）、饱和度、搭配
- **空间**：填充度、留白比例、元素分布均衡感
- **细节**：整体丰富度、某些部分过度细节 vs 简化
- **构图**：元素排列、视线引导、焦点位置
- **涂改**：擦除、覆盖、反复修改的区域

注意：老年用户的手部颤抖可能是生理性手抖，非情绪紧张。

## 绘画过程（来自视频）

### 绘画顺序
- 先画人物 → 最先关注自我形象
- 先画雨 → 最先关注环境氛围
- 先画背景 → 先建立空间框架

### 绘画节奏
- 流畅连贯 / 时有停顿 / 反复修改 / 加速减速

### 表情变化（webcam）
- 开始阶段：专注、犹豫、微笑
- 中间阶段：放松、皱眉、沉浸
- 结尾阶段：满意、审视、微笑、疲惫
