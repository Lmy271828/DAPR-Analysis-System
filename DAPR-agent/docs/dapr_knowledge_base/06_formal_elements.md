# DAPR 艺术元素分析维度详解

> 本文件提供 DAPR 绘画的详细观察维度，供系统在分析画面时参考。所有描述均为**中性艺术观察**，不附带心理诊断含义。

## 一、画面氛围元素

### 1.1 雨量表现

| 观察角度 | 描述示例 |
|---------|---------|
| 雨滴/雨线密度 | 稀疏的雨丝、中等的雨点、密集的雨幕 |
| 雨的方向 | 垂直下落、斜向飘洒、旋风式旋转 |
| 雨的形态 | 点状雨滴、线状雨丝、片状雨幕、不规则泼洒 |
| 动态感 | 静止的雨、飘动的雨、猛烈的暴雨 |
| 覆盖范围 | 局部降雨、画面半幅降雨、全画面降雨 |

**刘伟全息体系的重要发现**：压力大小与雨点长短无关，与雨点数量和密度呈正相关。

### 1.2 云层描绘

| 类型 | 描述 |
|------|------|
| 晴朗无云 | 天空留白或使用浅色 |
| 薄云 | 淡淡的云层，光线仍可穿透 |
| 厚云/乌云 | 浓密的云层，可能覆盖大部分天空 |
| 黑云 | 极深的色调 |
| 云层中的光线 | 云缝中有光线透出 |

### 1.3 闪电与风

- 闪电的形态：直线形、枝状、未画完的线条
- 风的表达：飘动的衣物、倾斜的雨线、飞散的物体
- 声音的象征：闪电通常伴随雷声，但画面中无法表现

### 1.4 地面与积水

| 元素 | 观察角度 |
|------|---------|
| 地面线 | 有无、清晰/模糊、完整/中断 |
| 积水/水坑 | 有无、大小、倒影 |
| 地面材质 | 泥土、水泥、草地、未指定 |
| 地面的干湿 | 干燥、湿润、积水明显 |

**注意**：无地面线在部分文献中被讨论为"不安全感"的可能表现，但系统应仅描述为"画面中没有出现地面线"，不做推断。

## 二、人物表现

### 2.1 人物完整性

| 程度 | 描述 |
|------|------|
| 全身像 | 头部、躯干、四肢均完整 |
| 半身像 | 只画到腰部或胸部 |
| 特写 | 仅画头部或脸部 |
| 不完整 | 缺少某些身体部位 |

**注意**：只画脸部的比例较高时，在日本研究中被发现与回避型思维相关，但系统不应向用户传达此关联。

### 2.2 人物姿态

| 姿态 | 描述 |
|------|------|
| 站立 | 直立、挺拔、放松 |
| 行走 | 迈步、向前、方向明确 |
| 奔跑 | 动态感强、肢体伸展 |
| 蹲坐 | 低姿态、收缩 |
| 躺卧 | 水平、与地面接触 |
| 其他 | 跳舞、跳跃、跌倒等 |

### 2.3 面部表情

| 表情 | 描述 |
|------|------|
| 微笑 | 嘴角上扬 |
| 平静/无表情 | 面部线条中性 |
| 忧愁 | 眉头微蹙、嘴角下垂 |
| 哭泣 | 眼泪、面部扭曲 |
| 未画面部 | 背影、侧脸无表情、面部留白 |

**文化注意**：不同文化中微笑的基线比例差异巨大（泰国 64% vs 墨西哥 20%），不应将微笑与否作为跨文化比较的依据。

### 2.4 人物在画面中的大小和位置

| 维度 | 观察 |
|------|------|
| 大小 | 占据画面的比例（很小/<10%、小/10-25%、中等/25-50%、大/>50%）|
| 位置 | 画面中央、偏左、偏右、上方、下方 |
| 与画面的关系 | 是否有足够的"呼吸空间"、是否被边缘切割 |

## 三、防护与伴随元素

### 3.1 防护工具

| 类型 | 有效性观察 |
|------|-----------|
| 雨伞 | 大小是否足够、是否正确使用（遮挡角度）、是否破损 |
| 雨衣 | 是否覆盖全身、颜色、细节 |
| 雨帽 | 类型（宽檐帽、斗笠等）|
| 雨靴 | 有无、样式 |
| 建筑物/屋檐 | 大小、人物是否在其下、距离 |
| 树木/植物 | 密度、是否能提供遮蔽 |
| 其他创意防护 | 任何用户自创的避雨方式 |

**重要**：
- 有防护工具但未使用 → 仅描述"画了伞但人物未在伞下"
- 防护工具破损 → 仅描述"伞面有缺口/破损"
- 不推断"防护无效 = 应对资源不足"

### 3.2 同伴与动物

| 类型 | 观察 |
|------|------|
| 其他人 | 人数、关系（共撑一伞、前后行走、距离远近）|
| 动物 | 类型（狗、猫、鸟等）、与人的关系 |
| 想象中的伙伴 | 任何非现实的存在 |

### 3.3 积极元素

| 元素 | 描述 |
|------|------|
| 彩虹 | 颜色数量、完整性、位置 |
| 阳光/光线 | 来源、照射方向、穿透云层 |
| 植物/花朵 | 类型、状态（盛开、发芽）|
| 温暖的色彩 | 画面的整体色调倾向 |
| 其他积极符号 | 星星、月亮、鸟类飞翔等 |

## 四、形式元素观察（FEATS 维度参考）

Formal Elements Art Therapy Scale（艺术治疗正式元素量表）提供以下观察角度：

| 维度 | 观察内容 |
|------|---------|
| **线条质量** | 力度（重/轻/中等）、连贯性（流畅/断裂/颤抖）、方向（水平/垂直/曲线/斜线）|
| **色彩使用** | 色调（暖/冷/中性）、饱和度、色彩搭配、色彩与情绪的对应 |
| **空间运用** | 画面填充度、留白比例、元素分布的均衡感 |
| **细节程度** | 整体丰富度、某些部分的过度细节 vs 其他部分的简化 |
| **构图** | 元素的排列方式、视线引导、焦点位置 |
| **涂改痕迹** | 擦除、覆盖、反复修改的区域 |

**注意**：线条质量中的"颤抖"在老年用户中可能是生理性手抖，不应解读为情绪紧张。

## 五、绘画过程观察（来自视频分析）

### 5.1 绘画顺序

记录用户先画什么、后画什么：
- 先画人物 → 可能最先关注自我形象
- 先画雨 → 可能最先关注环境氛围
- 先画地面/背景 → 可能先建立空间框架

**注意**：绘画顺序受多种因素影响（习惯、构图策略、突发灵感），不应过度解读。

### 5.2 绘画节奏

| 特征 | 描述 |
|------|------|
| 流畅连贯 | 一笔接一笔，无明显停顿 |
| 时有停顿 | 绘画过程中多次停顿和思考 |
| 反复修改 | 某些部分画了擦、擦了画 |
| 加速/减速 | 不同阶段的绘画速度变化 |

### 5.3 面部表情变化（ webcam 视频）

| 阶段 | 可能的观察 |
|------|-----------|
| 开始 | 专注、犹豫、微笑 |
| 中间 | 放松、皱眉、沉浸在创作中 |
| 结尾 | 满意、审视、微笑、疲惫 |

## 六、系统输出原则

在分析以上所有元素时，系统应遵循：

1. **描述先于解释**：先描述"看到了什么"，再温和地分享"这让人想到..."
2. **不确定语气**：使用"似乎" "可能" "让人感受到" 而非"说明" "反映"
3. **多可能性并存**：同一元素可能有多种解读，系统应呈现为"这可以是这样，也可以是那样"
4. **邀请用户参与**：以问题结束每个观察，邀请用户分享自己的理解
